# US Road Accident Analytics: Unveiling Trends and Insights from US Road Data

## Project Overview

This Business Intelligence and Data Analytics project analyzes the US Accidents dataset from Kaggle to identify accident trends, risk factors, accident hotspots, and severity patterns across the United States.

The project combines Exploratory Data Analysis (EDA), Machine Learning, Predictive Analytics, and Power BI visualization to generate actionable insights that support policymakers, road safety authorities, and emergency response teams.

---

## Objectives

- Analyze accident trends across time, weather conditions, and locations.
- Identify factors influencing accident severity.
- Detect high-risk accident hotspots.
- Predict accident severity and accident counts using machine learning.
- Develop interactive dashboards for data-driven decision-making.

## My Contributions

### Data Collection & Preprocessing

- Retrieved and analyzed the US Accidents dataset from Kaggle.
- Handled missing values and outliers using IQR and Z-score techniques.
- Performed feature engineering including time-based features (hour, day, month) and weather classifications.
- Prepared datasets for machine learning and visualization.

### Exploratory Data Analysis (EDA)

- Conducted univariate and bivariate analysis to identify accident patterns.
- Analyzed accident trends based on time of day, weather conditions, and geographical regions.
- Performed correlation analysis and time-series analysis.
- Applied geospatial analysis to identify accident hotspots and critical risk areas.

## Model Evaluation

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score

- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Percentage Error (MAPE)

---

## Results

- Random Forest achieved the highest classification accuracy of **90.73%**.
- Random Forest Regressor delivered the best performance for accident count prediction.
- Identified significant risk factors including weather conditions, time of day, and road characteristics.
- Generated actionable recommendations to improve road safety and accident prevention strategies.
